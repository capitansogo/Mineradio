# Mineradio (RU)

**Mineradio** — иммерсивный музыкальный плеер для Windows на Electron: не просто проигрыватель, а визуальная сцена вокруг трека. Погодное радио, поиск и воспроизведение, сцена с текстами песен, визуализация частицами из обложки, кинематографичные кадры под ритм и 3D-полка плейлистов.

Это **русскоязычная адаптация** оригинального проекта [XxHuberrr/Mineradio](https://github.com/XxHuberrr/Mineradio): добавлена **Яндекс Музыка** и **полная русификация интерфейса** с переключателем языка.

## Возможности

- 🎵 **Яндекс Музыка** — поиск, воспроизведение (с полным визуалом), вход через OAuth, плейлисты и лайки («Мне нравится»)
- 🌐 **NetEase Cloud Music** и **QQ Music** — поиск, вход, плейлисты, подкасты (из оригинала)
- 🇷🇺 **Русский интерфейс** с переключателем **РУ / 中** (выбор запоминается)
- 🌦 Погодное радио (Open-Meteo): очередь под текущую погоду, город и настроение
- ✨ Визуальные режимы: частицы из обложки, сцена с текстами, кинокадры под ритм
- 🎚 DIY-пульт визуала: цвета, плотность частиц, глубина резкости, текст на рабочем столе, обои
- 🗄 3D-полка плейлистов по правому клику
- 🔄 Проверка обновлений через GitHub Releases

## Запуск из исходников

```bash
npm install
npm start            # запуск в Electron
node --check server.js
```

Главная логика фронтенда — в `public/index.html`. Локальный API, музыкальные источники и проверка обновлений — в `server.js`. Точка входа Electron — `desktop/main.js`.

## Сборка установщика Windows

**Локально:**
```bash
npm run build:win    # NSIS-установщик в dist/
```

**Через GitHub Actions:** workflow `.github/workflows/build-windows.yml` собирает установщик в облаке.
- **Вручную:** вкладка *Actions* → *Build Windows installer* → *Run workflow* (артефакт `Mineradio-Setup` можно скачать со страницы запуска).
- **Релизом:** запушь тег вида `v1.1.1` — установщик соберётся и автоматически прикрепится к GitHub Release.

```bash
git tag v1.1.1
git push origin v1.1.1
```

## Вход в Яндекс Музыку

Кнопка аккаунта → вкладка **Яндекс** → «Войти» открывает окно авторизации Яндекса (OAuth). Токен сохраняется локально в `.ya-token` (в репозиторий не попадает). Для разработки можно положить токен в переменную окружения `YANDEX_MUSIC_TOKEN`.

## Третьи музыкальные платформы

Mineradio не является официальным клиентом Яндекс Музыки, NetEase, QQ Music или любой другой платформы и не аффилирован с ними.

Интеграции используют **неофициальные API** и предназначены только для личного использования с **собственным аккаунтом** пользователя. Соблюдайте пользовательские соглашения и правила соответствующих сервисов. Проект не обходит оплату, подписку или DRM и не распространяет музыкальный контент.

## Данные и приватность

Cookie/токены логина, история поиска, кастомные обложки и тексты, кэш ритм-анализа хранятся только локально (в каталоге пользователя или localStorage) и не коммитятся в репозиторий. Файлы `.cookie`, `.qq-cookie`, `.ya-token` в `.gitignore`.
